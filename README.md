# Personal Portfolio Website

A modern, responsive, and interactive portfolio website built to showcase my projects, skills, experience, and journey as a Full-Stack Developer. This portfolio serves as my digital presence, highlighting the solutions I have built and my passion for creating software that solves real-world problems.

## Live Demo

Portfolio Website: https://portfolio-woad-seven-31.vercel.app/

---

## Overview

This portfolio was designed and developed to provide recruiters, collaborators, and fellow developers with a clear view of my technical skills, featured projects, professional background, and career aspirations.

The website focuses on clean design, smooth animations, responsiveness, accessibility, and performance while maintaining a professional user experience across all devices.

## Installation

Clone the repository:

```bash
git clone https://github.com/Daniella-fejiro/portfolio.git
```

Navigate to the project directory:

```bash
cd portfolio
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Open your browser and visit:

```bash
http://localhost:3000
```
